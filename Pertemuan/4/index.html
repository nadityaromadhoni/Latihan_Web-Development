Sedang Latihan Table HTML

Table HTML adalah sebuah elemen markup pada bahasa pemrograman HTML yang digunakan untuk membuat tabel atau matriks dengan baris dan kolom,
Tabel HTML biasanya digunakan untuk menampilkan data dalam bentuk tabular yang terstruktur dan mudah dibaca oleh pengguna.
