Sedang Latihan List HTML

Unordered HTML

Unordered HTML adalah jenis tampilan daftar yang menggunakan tanda bulat
atau lingkaran untuk setiap elemen, biasanya digunakan untuk membuat daftar
yang tidak memiliki urutan atau prioritas tertentu.

  • Apel
  • Pisang
  • Jeruk

Ordered HTML

Ordered HTML adalah jenis tampilan daftar yang menggunakan nomor atau angka untuk setiap elemen
Biasanya digunakan untuk membuat daftar yang memiliki urutan atau prioritas tertentu.

  1. Buka laptop
  2. Jalankan browser
  3. Masukkan URL website

Definition List HTML

Definition List pada HTML adalah jenis tag yang digunakan untuk membuat
daftar definisi dengan menggunakan tag dl untuk membungkus daftar tersebut, tag dt untuk menandai judul
atau istilah, dan tag dd untuk menandai definisi dari masing-masing istilah tersebut..

HTML
HyperText Markup Language, bahasa markup yang digunakan untuk membuat halaman web.
CSS
Cascading Style Sheets, bahasa stylesheet yang digunakan untuk mengatur tampilan dari halaman web yang dibuat dengan HTML.
